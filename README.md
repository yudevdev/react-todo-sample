# React + TypeScript + MUI TODOアプリ

このリポジトリは、React, TypeScript, Material-UI (MUI), Vite を使用したシンプルな TODO アプリのサンプルプロジェクトです。

## セットアップ

### 1. リポジトリをクローン
```sh
git clone <リポジトリURL>
cd <プロジェクトフォルダ名>
```

### 2. 必要なパッケージをインストール
```sh
npm install
```

### 3. 開発サーバーを起動
```sh
npm run dev
```

ブラウザで `http://localhost:5173/` にアクセスすると、アプリが表示されます。
